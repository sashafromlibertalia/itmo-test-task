## Download project
```bash
git clone https://github.com/sashafromlibertalia/itmo-test-task.git
```

## Restore all dependencies
```bash
npm install
```


## Run in dev mode

```bash
npm run dev
```

## Build project
```bash
npm run build
```


## Run tests
```bash
npm run test
```
